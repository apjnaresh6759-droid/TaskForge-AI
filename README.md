# TaskForge-AI 🚀

An intelligent, self-contained Python-based task manager built with custom deadline generation, automated tagging, and project breakdown tools.

## 🛠️ Usage

```bash
# Generate task breakdown for a project
python taskforge.py generate "Build a mobile app"

# List current active tasks
python taskforge.py list

# List all tasks including completed ones
python taskforge.py list --all

# Mark a task as done
python taskforge.py done 1
